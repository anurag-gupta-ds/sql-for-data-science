The objective of this repository is to develop practical skills in querying databases, data aggregation, joins, subqueries, window functions, and extracting meaningful insights using SQL.

## 🎯 Goals

- Understand relational database structure and design
- Write queries to create, insert, update, and delete data
- Filter, sort, and limit query results
- Perform data aggregation using GROUP BY, HAVING, and ROLLUP
- Combine multiple tables using Joins (Inner, Left, Right, Self)
- Maintain data integrity using Constraints and Foreign Keys
- Apply CASE statements for conditional logic
- Use CTEs to simplify complex queries
- Apply Window Functions for ranking and comparative analysis
- Manage transactions using COMMIT and ROLLBACK
- Automate actions using Triggers and Stored Procedures
- Optimize queries using Indexes and Views
- Solve real-world business problems using SQL

## 📈 Techniques Applied

### 🗄️ Core SQL
- Database & Table Creation
- INSERT, UPDATE, DELETE
- SELECT, WHERE, ORDER BY, LIMIT
- Constraints (UNIQUE, CHECK, NOT NULL)
- ALTER TABLE, RENAME TABLE
- Aggregate, String, and Date Functions
- GROUP BY, HAVING, and ROLLUP

### 🔗 Relationships & Combining Data
- Inner, Left, Right Joins
- Self Join
- Foreign Keys and Referential Integrity
- UNION and UNION ALL

### 📊 Advanced SQL
- Subqueries
- CASE Statements
- Common Table Expressions (CTEs)
- Window Functions
  - Aggregate (SUM, AVG, MAX, MIN with PARTITION BY)
  - Ranking (ROW_NUMBER, RANK, DENSE_RANK)
  - LAG and LEAD

### ⚙️ Database Management & Automation
- Indexes
- Views
- Transactions (COMMIT, ROLLBACK, Autocommit)
- Stored Procedures
- Triggers
